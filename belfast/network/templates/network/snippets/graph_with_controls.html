Network Graph Help

Nodes in the graph are positioned based on the strength of connection between individuals or organizations, with an attempt to avoid overlapping labels for readability.

You can change the appearance of the graph in a number of different ways in order to explore its content. Clicking on any individual node will fix it in place (this is indicated by a black outline) and you can drag nodes to different positions on the graph. Doing this may help you see which individuals are connected to one another. You can make the nodes float freely again by double-clicking on them.

You can resize nodes based on available network properties, and you can turn the labels on or off, making it easier to identify the authors or to see the connections between them. When labels are turned off, you can hover over a particular node on the graph and see the name of the person it represents. You can also set the labels to appear only on nodes that are a particular size.

Click on a node or node label for more information
